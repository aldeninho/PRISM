##  What This Project Does

MoneyPrinterV2 is an automation-based project that focuses on generating and publishing online content without manual effort. It creates content and posts them to platforms such as YouTube and Twitter.


---

## Repository Structure

The repository is divided into different components, each responsible for a specific function:

* `src/` -> Contains the core logic and main program
* `scripts/` -> Handles automation tasks like posting and scheduling
* `assets/` -> Stores images, videos, and other media resources
* `fonts/` -> Includes fonts used during content generation
* `.github/` -> Contains configuration files for workflows and version control

---

##  Tools and Technologies

This project is primarily built using Python and relies on several libraries for automation and media processing.

* Language: Python
* Libraries: MoviePy (video editing), Selenium (automation), Coqui TTS (text-to-speech)
* External Tools: Social media APIs (YouTube, Twitter)
* Scheduling: CRON jobs for automated execution
